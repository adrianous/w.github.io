更换服务器和域名总结

2020-10-26

  • 购买腾讯云 服务器 和 域名
    • 选择最便宜的，方便备案和实名认证。
    • 登录服务器，配置密钥登录（需要添加腾讯云生成的密钥，后续可以添加本机公钥，我是在 git 提交时添加的）
    • 申请 SSL，配置 SSL，配置 http 强制跳转 https
  • 使用 iTerm2 Profile快速打开 ssh 登录服务器
  • mac 安装 ftp 被包含在 inetutils 内，brew 无法直接安装 ftp
    • 需要 sftp 连接，未添加本地公钥时，需要指定密钥路径
    • 使用 ftp 基本命令，put & get
  • 安装 nginx
    • nginx 介绍：异步框架的网页服务器（目前作为web server来处理静态资源）
    • 配置 nginx （网页根目录，域名）
  • 安装 git
    • 初始化 git，添加 git 用户，添加 git 用户密钥登录
    • 限制 git 用户shell登录权限
    • 添加 git-hooks
  • 修改原来 blog 配置信息
    • 修改 deploy 配置
    • blog git 还是放在 github 了
    • 原来的 theme 丢失了 重新下载并修复

此次重启 Blog，为了记录 机器学习之旅，目标：推荐和预测。
